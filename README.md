# Tuist Practice
Tuist로 프로젝트 생성 및 모듈화 연습용 저장소입니다.

<br>

## 프로젝트 빌드

1. mise를 통해 Tuist를 설치한 상태여야합니다. [Tuist 설치 방법](https://docs.tuist.io/guide/introduction/installation.html)
2. git clone
3. 프로젝트 경로에서 tuist install
4. tuist generate

<br>

## 배운 점

1. Tuist로 프로젝트 생성하는 방법
2. Tuist로 모듈화하는 방법
